About ListOnGo

ListOnGo is a smart reminder app built in Java for Android. It helps you never forget grocery or essential items while shopping by letting you prepare, store, and manage shopping lists—and reminding you right when you need it.

Features

  • User authentication with email/password and OTP verification
  • Top toolbar displaying username, credit balance, and quick access to settings, reminders, and admin checks
  • Browse and search items by categories like Soft Drinks, Medicine, Fashion, and more
  • Add new products with details like title, nickname, description, price, category, image; submission cost is credit-based and requires admin approval
  • Keep track of previously created lists by date, total cost, and product details
  • Profile page showing user ID, name, registered email, with ability to logout
  • Settings options including notifications, biometric login, theme toggle, credit purchase, contact support, privacy policy, about page, feedback, and clear list
  • Admin panel access for pending approvals, approved items, and product edits (if user has admin privileges)
  • Seamless backend integration using Java Spring Boot and PostgreSQL

How It Works

Think of ListOnGo as your personal shopping assistant. Create a list, include what you need, and let the app remind you while shopping—no more forgotten essentials.

Tech Stack

  • Android frontend: Java with clean XML layouts
  • Backend API: Spring Boot (Java – full stack)
  • Database: PostgreSQL
